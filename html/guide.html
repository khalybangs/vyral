An Author Account is an account that can easily achievable by simply upgrading the average user account. This account allows users to create contents that can only be available to their subscribers.

With this account upgrade, a user can:

  • set a subscription fee before a user can have access to the account (or can simply leave it as free, which is the default)
  • choose certain categories an individual wants their contents to be categorized by
  • can upload books
  • can upload posts as the average user as articles or contents for your subscribers

All of these functionalities are achievable for free. We are only interested in 10% interest from any of a user's new subscription paid. All earnings, interests and charges made on an author account can be managed through settings.

Other charges include payment and withdrawal charges which are both 1.55% each. Totaling: 3.10%.

A user can upgrade to this account via settings or you can click here.
